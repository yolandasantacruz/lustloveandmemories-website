# lustloveandmemories-website

This repository contains the source code for the website [Lust, Love, and Memories](https://www.lustloveandmemories.com), dedicated to the book series by author Yolanda Santa Cruz.

## The Collection

### "Lust, Love, and Memories (Volume I)"
"Lust, Love, and Memories" is a compilation of poems that focuses on all stages of romantic relationships, from the desire and excitement felt at the very beginning, to the aftermath of them.

### "The Longest Nights (Volume II)"
Following a confessional poetry style, "The Longest Nights" dives into the healing journey the author went through after experiencing a romantic loss. With each chapter dedicated to the five most renowned stages of grieving, the collection features poems that range from complete denial of the present to the acceptance that comes only after making peace with the past.

## About the Author
Yolanda Santa Cruz is an artist and writer who believes in the therapeutic power of art. She views art as a catalyst that can heal and uplift both the creators and those who connect with it. Her creative journey has led to the writing of two books centered around romantic relationships and everything they encompass. The poems delve into deeply personal territory, serving as a window into moments where emotions ran so deep that they needed expression, documenting the evolution of passions and heartbreaks encountered along the way.

Her work has been featured in various publications and anthologies, including "Recuerdo Incorrumpible, Vol.2" (Spanish edition) and Visor Magazine. She has also participated in various festivals and residencies, such as The Mauser Foundation, Hypatia in the Woods, and World of Co.

Contact and further work can be explored on Instagram at [@lustloveandmemories](https://instagram.com/lustloveandmemories).

### Setup

Run `pnpm install` to install the application dependencies.

### Development

Run `pnpm start` for a dev server. Navigate to `http://localhost:5173/`. The application automatically reloads if you change any of the source files.

### Build

Run `pnpm run build` to build the client/server project. The client build artifacts are located in the `dist/analog/public` directory. The server for the API build artifacts are located in the `dist/analog/server` directory.
